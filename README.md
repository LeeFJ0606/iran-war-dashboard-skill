# 伊朗战争看板 · 多角色态势模拟

一个输入“新闻链接”、输出“22个角色观点与下一步措施”的交互式网页与同名 Skill。覆盖伊朗最高层、伊朗派系、美以阵营、海湾与地区国家、中国、俄罗斯，以及“伊朗民众/美国民众/以色列民众”三类社会公众角色。

## 在线预览
- GitHub Pages（项目页）：`https://LeeFJ0606.github.io/iran-war-dashboard-skill/`
- 若首次访问 404，请稍等 1–2 分钟等待构建完成，或在仓库 Settings → Pages 手动启用：
  - Build and deployment → “Deploy from a branch”
  - Branch 选择 `gh-pages`，Folder 选择 `/`，保存

## 使用方式
- 打开网页后在输入框粘贴新闻链接，点击“分析”
  - 示例：`https://baijiahao.baidu.com/s?id=1858457899240027301`
- 页面将只读抓取公开文本，生成：
  - 事件要点摘要
  - 22个角色的“观点”“下一步措施”“标签”
  - 分组筛选、复制全部、导出 JSON

## 交互说明
- 分组筛选：按“伊朗最高层/伊朗派系/美以阵营/海湾国家/地区国家/全球主要国家/社会公众”等分组展示
- 导出 JSON：一键下载结构化输出，便于二次分析
- 复制全部：整页角色输出文本复制到剪贴板
- 重置：清空输入与分析结果

## 角色列表
- 伊朗：新任最高领袖、总统、强硬派高层、亲美派高层、中立派高层、民众
- 美以阵营：美国总统、以色列总统、美国民众、以色列民众
- 海湾与地区：沙特、阿联酋、巴林、卡塔尔、阿曼、科威特、伊拉克、叙利亚、也门、黎巴嫩（伊朗盟友）
- 全球主要国家：中国、俄罗斯

## 数据来源与安全
- 只读抓取公开页面文本（前端代理），不进行登录、提交或修改操作
- 输出为策略模拟与预测，不代表事实或官方立场
- 对极端舆论进行稳健处理，避免过度放大

## 本地预览
- 克隆仓库后，任意静态服务器或浏览器直接打开 `index.html` 即可
- 例如：
  ```
  python3 -m http.server 8000
  # 访问 http://localhost:8000/
  ```

## Skill 说明
- 技能文件：`SKILL.md`
- 触发条件：用户要求进行多角色态势分析或提供新闻链接时调用
- 输出：摘要 + 22角色观点/措施/标签（结构化 JSON）

## 目录结构
- `index.html`：网页入口
- `assets/style.css`：样式
- `assets/app.js`：逻辑（抓取、解析、角色生成、交互）
- `SKILL.md`：Skill 说明
- `README.md`：本文件
- （可选）`.nojekyll`：禁用 Jekyll（便于静态资源载入）
