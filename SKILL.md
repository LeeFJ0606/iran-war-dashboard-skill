# 伊朗战争看板 Skill

本技能用于对指定新闻链接进行只读抓取与文本解析，依据战争策略、国际关系、伊朗内部派系、人性与舆论、石油与宗教等维度，生成22个角色（含新增“伊朗民众”“美国民众”“以色列民众”）的“观点”与“下一步措施”，并附带事件摘要与标签。

## 适用场景（触发条件）
- 用户提供新闻链接并要求进行多角色态势分析与行动预测
- 用户提出“伊朗-美以”相关的冲突动态，希望得到结构化的角色输出
- 需要快速形成对事件的摘要与角色横向比较视图

## 输入
- `news_url`：字符串，公开可访问的新闻页面链接

## 输出
- `summary`：事件要点摘要（多行文本）
- `roles`：数组，包含以下字段：
  - `id`：角色标识
  - `name`：角色名称
  - `group`：分组（如“伊朗派系”“海湾国家”“社会公众”等）
  - `thought`：角色对事件的观点
  - `actions`：角色预计的下一步措施（若干条）
  - `tags`：风险与主题标签（如“霍尔木兹”“航运风险”“高层遇害”“油价压力”“民防”等）

## 步骤
1. 抓取：以安全只读方式抓取 `news_url` 页面文本（可使用只读代理，如将 `https://` 转换为 `https://r.jina.ai/http://...` 以获取纯文本），若抓取失败输出通用摘要与保守策略。
2. 构建上下文信号（示例）：`hormuz`（霍尔木兹）、`tanker`（油轮受袭/航运风险）、`leaderKilled`（高层遇害/斩首）、`israel`、`usa`、`iran`、`oilDep`（石油与财政依赖）、`pipeline`（替代管道能力）、`escalation`（升级强度）、`market`（油价与通胀）、`sect`（宗教相关）、`media`（舆论与社媒）。
3. 摘要：基于信号生成多条要点摘要，覆盖海上通道风险、指挥链承压、联盟对抗、能源依赖与替代能力等。
4. 角色生成：为每个角色生成“观点”“措施”“标签”。核心角色集：
   - 伊朗新任最高领袖、伊朗总统、伊朗强硬派高层、伊朗亲美派高层、伊朗中立派高层
   - 美国总统、以色列总统
   - 海湾与地区：沙特、阿联酋、巴林、卡塔尔、阿曼、科威特、伊拉克、叙利亚、也门、黎巴嫩（伊朗盟友）
   - 全球主要国家：中国、俄罗斯
   - 社会公众：伊朗民众、美国民众、以色列民众
5. 风险刻度：结合 `escalation` 与 `hormuz`/`oilDep` 等信号为标签附加强度，如“升级:高/中/低”“能源:高/中/低”“高警戒”“油价压力”“物资压力”等。
6. 输出：返回摘要与角色数组，支持后续渲染或导出。

## 安全与限制
- 本技能仅进行策略模拟与预测，不代表事实结论或官方立场
- 只读抓取公开页面，不进行登录、提交或修改操作
- 对不完整或偏激的舆论内容进行稳健处理，避免过度放大极端观点
